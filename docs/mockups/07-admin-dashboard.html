Design Decisions - Mockup 07: Admin Dashboard

  • Stat cards use Oswald for values to create visual hierarchy and a bold, authoritative feel consistent with the RLC brand.
  • "Needs Attention" section uses amber/warning palette to draw the eye to actionable items without signaling an error state. Each item has a direct action link.
  • Activity feed uses color-coded dots and icons to enable at-a-glance scanning -- green for new members, blue for campaigns, gold for donations, purple for events, red for expirations.
  • Sidebar is fixed-position with collapsible mobile behavior using a hamburger toggle. Navigation is grouped by function (Overview, Manage, Content, Settings).
  • Responsive breakpoints: 4-col stat grid collapses to 2-col at 1024px, 1-col at 375px. Two-column content grid stacks at 1024px. Sidebar hides off-screen on mobile.
  • Quick Actions use dashed borders to signal "create new" affordance, transitioning to solid blue borders and blue backgrounds on hover for clear interactivity.
