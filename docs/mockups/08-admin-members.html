Design Decisions - Mockup 08: Admin Members List

  • Tier badges use gradient backgrounds with borders to create visual distinction and hierarchy: Lifetime (gold), Patron (purple), Sustaining (blue), Regular (gray), Student (green).
  • Status badges use semantic colors: Current (green), Expired (red), Grace Period (amber), New (blue). These match standard admin UI conventions for instant recognition.
  • Bulk action bar appears as a blue overlay at the top of the table when checkboxes are selected, making the multi-select mode visually obvious and providing contextual actions (Export, Email, Change Status).
  • Search input has a visual search icon embedded via CSS background-image SVG to avoid icon library dependencies. Filter dropdowns use custom caret styling.
  • Sort indicators on column headers (triangle arrows) show current sort state. Active sort is highlighted in blue. All headers are clickable for sort interaction.
  • Column visibility toggle (gear icon) allows admins to customize which columns are visible, useful for narrower viewports or specific workflows.
  • Table scrolls horizontally on mobile via overflow-x: auto, preserving all columns and data integrity on small screens.
